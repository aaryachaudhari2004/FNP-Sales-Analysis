The Sales Analysis Dashboard is an interactive business intelligence solution developed to analyze sales performance, customer purchasing behavior, product performance, seasonal demand, and geographical sales distribution. The dashboard enables management and stakeholders to monitor key business metrics in real time and make data-driven decisions to improve revenue generation and operational efficiency.

The dashboard consolidates sales data into a single visual interface, providing insights into revenue trends, order patterns, customer spending habits, product performance, and city-wise demand. Users can dynamically filter data using Order Date, Delivery Date, and Occasion filters for detailed analysis.
